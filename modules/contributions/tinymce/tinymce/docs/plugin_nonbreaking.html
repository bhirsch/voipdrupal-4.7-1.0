Plugin: nonbreaking

This plugin a button insert nonbreaking space entities &nbsp; at the current carent location.

Settings

This plugin has just one setting "nonbreaking_force_tab" this enables you to force TinyMCE to insert three &nbsp; entities when the user presses the tab key.

Installation Instructions

  • Add plugin to TinyMCE plugin option list example: plugins : "nonbreaking".
  • Add the nonbreaking button name to button list, example: theme_advanced_buttons3_add : "nonbreaking".

Initialization Example

tinyMCE.init({
	theme : "advanced",
	mode : "textareas",
	plugins : "nonbreaking",
	theme_advanced_buttons3_add : "nonbreaking",
	nonbreaking_force_tab : true
});
